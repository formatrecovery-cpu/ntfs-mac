## 易用性

虽然设置过程有些复杂，但赤友NTFS助手 的易用性是无与伦比的。内核安装完成后，你就可以使用赤友NTFS助手软件来控制所有连接到 Mac 的 NTFS 磁盘。

你可以选择挂载这些磁盘，它们会出现在桌面上或访达窗口中。然后，你就可以像管理 APFS文件系统一样管理 NTFS 文件系统了。也就是说，你能获得与 APFS文件系统相同的速度和性能。

这种速度是我们在 ExFAT 文件系统上从未体验过的，即使在 macOS 的最佳设置下也不行。

## 高级磁盘控制

我们曾以为这是 macOS 上能获得的最先进的磁盘控制系统。然而，我们知道当时 NTFS 尚未得到支持。当我们看到赤友NTFS助手无缝支持的界面时，我们意识到自己错了。你在应用程序中获得的高级磁盘控制选项非常棒。

一旦挂载了NTFS格式磁盘，你就拥有了它的完全访问权限。你可以编辑文件，删除文件，更改文件，写入新文件等。还有一个选项可以将其他驱动器转换为NTFS文件系统，例如 ExFAT格式的磁盘。通常，你需要将硬盘插入 Windows 系统才能做到这一点，但使用赤友NTFS助手则无需如此。

我们可以用这个工具做一些很酷的事情。例如，我们可以轻松挂载一个 NTFS硬盘并将其设置为启动磁盘。只要该磁盘上有一个可用的操作系统，它就会出现在启动菜单中，并且我们可以通过 Bootcamp 选项加载 Windows。

为了真正了解赤友NTFS助手的性能优势，我们进行了进一步测试。我们比较了 ExFAT、原生 NTFS 只读访问和赤友NTFS助手之间的文件传输速度。我们的测试设备包括一台配备 M2 芯片的 MacBook Pro，以及各种格式化为 NTFS、ExFAT 和 APFS 的外置硬盘，并混合了小型、中型和大型文件。

读取大文件： 访问和打开大文件时，三种文件系统之间的差异很小，性能都足够。

总体而言，与 ExFAT 相比，赤友NTFS助手提供了显著的性能提升，尤其是在复制大文件或大量小文件时。虽然原生 NTFS 只读访问适合基本的文件操作，但赤友NTFS助手在写入密集型任务中表现出色。
